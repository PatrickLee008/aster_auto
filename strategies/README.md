# 交易策略说明

## 1. 刷量策略 (volume_strategy.py)

### 策略目标
通过卖出和买入相同价格和数量的现货来刷交易量，同时避免亏损。

### 策略逻辑
1. **获取价格区间**：查询指定现货的买一价格和卖一价格
2. **生成随机价格**：在买卖价差之间随机生成一个交易价格
3. **执行卖出**：按照生成的价格限价卖出指定数量
4. **立即买入**：按照相同价格限价买入相同数量
5. **风险控制**：如果买入在5秒内未成交，自动使用最优价格买入
6. **重复执行**：等待间隔时间后重复以上步骤

### 策略参数
- `symbol`: 交易对 (默认: "ASTERUSDT")
- `quantity`: 每次交易数量 (默认: "1.0")
- `interval`: 交易间隔时间 (默认: 10秒)
- `rounds`: 总交易轮次 (默认: 10次)

### 风险控制
- **买入超时**: 5秒内买入订单未成交则触发风险控制
- **紧急买入**: 使用卖一价格或市价单确保买入成交
- **价格偏差**: 限制最大价格偏差为1%

### 使用方法
1. **从主菜单运行**: 选择选项9 "执行刷量策略"
2. **直接运行策略**: 
   ```bash
   cd strategies
   python volume_strategy.py
   ```
3. **程序集成**:
   ```python
   from strategies.volume_strategy import VolumeStrategy
   
   strategy = VolumeStrategy(
       symbol="ASTERUSDT",
       quantity="1.0", 
       interval=10,
       rounds=10
   )
   strategy.run()
   ```

### 注意事项
1. 确保账户有足够的ASTER余额用于卖出
2. 确保账户有足够的USDT余额用于买入
3. 策略会在买卖价差之间生成价格，如果价差很小可能使用买一价格
4. 风险控制确保不会因为无法买回而产生损失
5. 建议先用小数量测试策略效果

### 策略效果
- **刷交易量**: 通过频繁交易增加交易量
- **无损交易**: 通过相同价格买卖避免价格损失
- **灵活参数**: 可根据需求调整交易频率和数量
